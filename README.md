# vcoin-cnm-2023

## Configuration

- config .env: VITE_DEV_BE=http://localhost:3001

## BE run mode

- npm start {be-PORT} {p2p-PORT} {MODE}
- be-PORT, p2p-PORT are ports for be and for p2p connection in order.

## Run FE

- npm run dev
